# To-Do List

Android app for efficient task management and planning. Designed to help users organize their daily activities and deadlines effectively.

## Features

-   Display a list of tasks.
-   Add tasks to the list.
-   Edit tasks from the list.
-   Delete tasks from the list.
-   Mark tasks as completed.
-   Generate reminders for tasks due today or overdue.
-   Import and export tasks.
-   Assign priorities to tasks.
-   Sort tasks by various criteria:
    -   Task name
    -   Priority
    -   Status (completed or not)
    -   Deadline

## Requirements

-   Android SDK (API level 21 or higher)
-   Android Studio
-   Gradle 7.0 or higher

## Installation and Setup

1.  Clone the repository:
    
    ```bash
    git clone https://github.com/lukasz-cz/to_do_list.git
    cd to_do_list
    
    ```
    
2.  Open the project in Android Studio.
    
3.  Build and run the application on an emulator or a physical Android device.
